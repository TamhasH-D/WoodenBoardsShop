# Тест-кейсы: Управление сущностями в Admin Frontend

## TC-AF-ENT-001: Просмотр списка покупателей
**Приоритет**: Критический  
**Теги**: `#frontend` `#admin` `#buyers` `#crud` `#critical`

### Предусловия
- Администратор авторизован в системе
- В базе данных есть записи покупателей

### Шаги выполнения
1. Перейти в раздел "Покупатели" (Buyers)
2. Дождаться загрузки списка
3. Проверить отображение данных

### Ожидаемый результат
- Отображается таблица с покупателями
- Показаны колонки: ID, Keycloak UUID, Статус онлайн, Последняя активность
- Работает пагинация (если записей больше 20)
- Есть кнопки для создания, редактирования, удаления

---

## TC-AF-ENT-002: Создание нового покупателя
**Приоритет**: Критический  
**Теги**: `#frontend` `#admin` `#buyers` `#create` `#critical`

### Предусловия
- Администратор авторизован в системе
- Открыт раздел "Покупатели"

### Шаги выполнения
1. Нажать кнопку "Создать" или "+"
2. Заполнить форму создания покупателя:
   - ID: оставить пустым для автогенерации или ввести UUID
   - Keycloak UUID: ввести валидный UUID
   - Статус онлайн: выбрать true/false
3. Нажать кнопку "Сохранить"

### Ожидаемый результат
- Форма успешно отправлена
- Новый покупатель появился в списке
- Отображается сообщение об успешном создании
- ID автоматически сгенерирован, если не был указан

---

## TC-AF-ENT-005: Создание продукта с использованием UUIDField
**Приоритет**: Критический  
**Теги**: `#frontend` `#admin` `#products` `#uuid` `#critical`

### Предусловия
- Администратор авторизован в системе
- Открыт раздел "Продукты"
- Существуют продавцы и типы древесины

### Шаги выполнения
1. Нажать кнопку "Создать продукт"
2. В поле ID нажать на текст "ID: будет сгенерирован автоматически"
3. Ввести собственный UUID или нажать кнопку генерации
4. Заполнить остальные поля:
   - Название: "Доски сосновые"
   - Описание: "Качественные сосновые доски"
   - Объем: 2.5
   - Цена: 15000
   - Продавец: выбрать из списка
   - Тип древесины: выбрать из списка
5. Нажать "Сохранить"

### Ожидаемый результат
- Продукт создан с указанным или сгенерированным UUID
- Все данные корректно сохранены
- Продукт отображается в списке
