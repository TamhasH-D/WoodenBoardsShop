# Тест-кейсы: Просмотр продуктов в Buyer Frontend

## TC-BF-BROWSE-001: Просмотр каталога продуктов
**Приоритет**: Критический  
**Теги**: `#frontend` `#buyer` `#products` `#catalog` `#critical`

### Предусловия
- Покупатель открыл сайт (авторизация не обязательна)
- В системе есть опубликованные продукты

### Шаги выполнения
1. Открыть главную страницу покупателя
2. Перейти в раздел "Продукты" или "Каталог"
3. Просмотреть список доступных продуктов
4. Проверить отображение информации о продуктах

### Ожидаемый результат
- Отображается список всех доступных продуктов
- Показаны: название, цена, объем, тип древесины, изображение
- Есть кнопки "Подробнее" и "Связаться с продавцом"
- Работает пагинация при большом количестве продуктов

---

## TC-BF-BROWSE-002: Детальный просмотр продукта
**Приоритет**: Критический  
**Теги**: `#frontend` `#buyer` `#products` `#details` `#critical`

### Предусловия
- Покупатель находится в каталоге продуктов
- Выбран конкретный продукт

### Шаги выполнения
1. Нажать на продукт или кнопку "Подробнее"
2. Просмотреть детальную информацию:
   - Полное описание
   - Характеристики древесины
   - Размеры досок (если есть)
   - Условия доставки
   - Контактная информация продавца
3. Просмотреть изображения продукта

### Ожидаемый результат
- Отображается полная информация о продукте
- Изображения загружаются корректно
- Есть возможность увеличить изображения
- Показана информация о продавце
- Доступны кнопки для связи с продавцом

## TC-BF-BROWSE-003: Поиск продуктов по названию
**Приоритет**: Высокий  
**Теги**: `#frontend` `#buyer` `#products` `#search` `#high`

### Предусловия
- Покупатель находится в каталоге продуктов
- В системе есть продукты с разными названиями

### Шаги выполнения
1. Найти поле поиска на странице
2. Ввести поисковый запрос: "сосна"
3. Нажать кнопку поиска или Enter
4. Просмотреть результаты поиска
5. Очистить поиск и проверить возврат всех продуктов

### Ожидаемый результат
- Отображаются только продукты, содержащие "сосна" в названии
- Поиск работает без учета регистра
- Результаты релевантны запросу
- Очистка поиска возвращает полный каталог

## TC-BF-BROWSE-008: Использование AI-анализатора досок
**Приоритет**: Высокий  
**Теги**: `#frontend` `#buyer` `#ai` `#board_analyzer` `#high`

### Предусловия
- Покупатель авторизован в системе
- AI-сервис доступен

### Шаги выполнения
1. Перейти в раздел "Анализатор досок" или "AI-помощник"
2. Загрузить изображение с досками
3. Ввести параметры:
   - Высота досок: 5 см
   - Длина досок: 250 см
4. Нажать "Анализировать"
5. Дождаться результатов анализа

### Ожидаемый результат
- Изображение успешно загружено
- AI определил количество досок на фото
- Рассчитан общий объем древесины
- Показаны координаты обнаруженных досок
- Результат можно использовать для поиска подходящих продуктов

## TC-BF-BROWSE-009: Сравнение продуктов
**Приоритет**: Низкий  
**Теги**: `#frontend` `#buyer` `#products` `#compare` `#low`

### Предусловия
- Покупатель просматривает каталог продуктов
- В каталоге есть несколько похожих продуктов

### Шаги выполнения
1. Выбрать 2-3 продукта для сравнения (чекбоксы)
2. Нажать кнопку "Сравнить выбранные"
3. Просмотреть таблицу сравнения:
   - Цены
   - Объемы
   - Типы древесины
   - Условия доставки
4. Выбрать лучший вариант

### Ожидаемый результат
- Отображается удобная таблица сравнения
- Все характеристики выровнены для легкого сравнения
- Выделяются лучшие значения (например, самая низкая цена)
- Можно перейти к любому из сравниваемых продуктов

## TC-BF-BROWSE-012: Обработка ошибок загрузки
**Приоритет**: Средний  
**Теги**: `#frontend` `#buyer` `#error_handling` `#medium`

### Предусловия
- Покупатель находится в каталоге
- Имитируется проблема с сетью или сервером

### Шаги выполнения
1. Отключить интернет или заблокировать API
2. Попытаться загрузить каталог продуктов
3. Проверить отображение ошибки
4. Восстановить соединение
5. Попытаться обновить страницу

### Ожидаемый результат
- Отображается понятное сообщение об ошибке
- Предлагается повторить попытку
- Интерфейс остается отзывчивым
- После восстановления соединения данные загружаются
- Есть возможность работы в офлайн-режиме (кэш)
