# Тест-кейсы: Управление продуктами в Seller Frontend

## TC-SF-PROD-001: Просмотр списка собственных продуктов
**Приоритет**: Критический  
**Теги**: `#frontend` `#seller` `#products` `#view` `#critical`

### Предусловия
- Продавец авторизован в системе
- У продавца есть созданные продукты

### Шаги выполнения
1. Открыть раздел "Мои продукты"
2. Дождаться загрузки списка
3. Проверить отображение продуктов

### Ожидаемый результат
- Отображаются только продукты текущего продавца
- Показаны: название, цена, объем, тип древесины, статус
- Есть кнопки для редактирования и удаления
- Работает пагинация при большом количестве продуктов

---

## TC-SF-PROD-002: Создание нового продукта
**Приоритет**: Критический  
**Теги**: `#frontend` `#seller` `#products` `#create` `#critical`

### Предусловия
- Продавец авторизован в системе
- Существуют типы древесины в системе

### Шаги выполнения
1. Нажать кнопку "Добавить продукт"
2. Заполнить форму создания:
   - Название: "Доски березовые 50x150"
   - Описание: "Качественные березовые доски для строительства"
   - Объем: 3.5 м³
   - Цена: 25000 ₽
   - Тип древесины: выбрать "Береза"
   - Возможность доставки: да
   - Место самовывоза: "г. Москва, ул. Примерная, 123"
3. Нажать "Создать продукт"

### Ожидаемый результат
- Продукт успешно создан
- Продавец автоматически привязан к продукту
- Продукт появился в списке "Мои продукты"
- Отображается сообщение об успешном создании

---

## TC-SF-PROD-003: Редактирование существующего продукта
**Приоритет**: Высокий  
**Теги**: `#frontend` `#seller` `#products` `#edit` `#high`

### Предусловия
- Продавец авторизован в системе
- У продавца есть созданный продукт

### Шаги выполнения
1. Найти продукт в списке "Мои продукты"
2. Нажать кнопку "Редактировать"
3. Изменить данные:
   - Цену: увеличить на 10%
   - Описание: добавить дополнительную информацию
   - Место самовывоза: изменить адрес
4. Нажать "Сохранить изменения"

### Ожидаемый результат
- Изменения сохранены в базе данных
- Обновленная информация отображается в списке
- Показано сообщение об успешном обновлении
- Дата обновления изменилась

## TC-SF-PROD-005: Загрузка изображения для продукта
**Приоритет**: Высокий  
**Теги**: `#frontend` `#seller` `#products` `#images` `#upload` `#high`

### Предусловия
- Продавец авторизован в системе
- Создается или редактируется продукт

### Шаги выполнения
1. В форме продукта найти раздел "Изображения"
2. Нажать "Загрузить изображение"
3. Выбрать файл изображения (JPG/PNG, до 5MB)
4. Дождаться загрузки
5. Проверить превью изображения

### Ожидаемый результат
- Изображение успешно загружено на сервер
- Отображается превью загруженного изображения
- Путь к изображению сохранен в базе данных
- Изображение доступно для просмотра покупателями

## TC-SF-PROD-006: AI-анализ досок на изображении
**Приоритет**: Высокий  
**Теги**: `#frontend` `#seller` `#products` `#ai` `#boards` `#high`

### Предусловия
- Продавец авторизован в системе
- Загружено изображение с досками
- AI-сервис доступен

### Шаги выполнения
1. После загрузки изображения нажать "Анализировать доски"
2. Ввести параметры:
   - Высота досок: 5 см
   - Длина досок: 300 см
3. Нажать "Запустить анализ"
4. Дождаться результатов AI-анализа

### Ожидаемый результат
- AI определил количество досок на изображении
- Рассчитан общий объем древесины
- Показаны координаты обнаруженных досок
- Данные автоматически заполнены в форме продукта
